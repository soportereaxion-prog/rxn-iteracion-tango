Módulo Operativo: Configuración Global (Directorio)

Propósito: Parametrizar las variables de entorno, credenciales y montajes de volúmenes necesarios para que el motor de sincronización se comunique con la instancia local de ReAxion conectada a Tango Gestión.

Diccionario de Variables Críticas

Parámetro Descripción y Efecto Operacional
Ruta CSV Volumen interno (Docker) o externo donde se depositan físicamente los lotes para su lectura. Ej: C:\TANGO\LadyApi / var/www/html/csv
Ruta Local API Endpoint base del intermediario ReAxion para la inserción en Tango. Ej: http://svrxn:17001
ID Empresa Identidad unívoca de la base de datos dentro del diccionario de la API (Ej: 306). Esto ruteará el alta a la empresa asignada.
Base de Datos Alias catalogado en SQL que hace match con el ID Empresa. Ej: LADY_WAY_SRL. Se autocompleta vía selector inteligente.
Tokens API Credenciales Bearer de autenticación provistas por ReAxion. Sin ellas el conector responderá código HTTP 401/403.
Numeradores Fiscales (Factura B / Ecommerce / Exportación). Rastrean el último talonario emitido. El sistema efectúa N+1. Si se rechazan o anulan comprobantes, estos números deben rebobinarse manualmente para no interrumpir la secuencia frente al facturador.

Consideraciones de Aislamiento

Zona Sensible (Solo Sysadmin): La alteración de estos datos detendrá instantáneamente el procesamiento si los mapeos de base de datos inter-empresa se corrompen. Evite guardar sin haber verificado la disponibilidad de los puertos mediante ICMP o Curl.
