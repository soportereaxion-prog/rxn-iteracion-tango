Módulo del Sistema

Reprocesar Rechazados

  • Versión: LadyApi001
  • Fecha de actualización: 20/05/2025
  • Propósito: Permitir el reprocesamiento de archivos que no se procesaron en el módulo de “Procesar datos”.

Descripción General

Este módulo permite al operador volver a procesar aquellos archivos que, por algún motivo, no pudieron ser procesados correctamente. Esto puede deberse a errores en los archivos CSV o a una acción realizada por el usuario, como haber ingresado al módulo de "Borrar pendientes" y presionado el botón de "Rechazar" por error o arrepentimiento.

Al entrar al módulo de “Reprocesar rechazados”, el sistema comenzará la lectura de archivos que fueron rechazados. Deberá presionar el botón “Procesar” para volver a ejecutar la acción de proceso. De ese modo, podrá visualizar los archivos pendientes de obtener CAE desde el sistema de TANGO GESTIÓN.

En ocasiones puede ocurrir que en los archivos CSV de comprobantes tengan datos que no estén registrados en Tango, como:

  • Factura que tiene un cliente que no está dado de alta en Tango.
  • Artículo que no está dado de alta en Tango.

Alertas y Validaciones

Este módulo posee las mismas alertas de validaciones que el módulo de “Procesar datos”.

Si procesa archivos de clientes que existen, el sistema lo detectará y mostrará un mensaje que dice:

“Se procesó el cliente con el DNI: xxxxxxxx ya existe”

La validación se hará a través del DNI del cliente.

Lo mismo sucederá con artículos; el sistema lo detectará y mostrará un mensaje que dice:

“No hay artículos para ingresar, sin embargo, el artículo ya existe en la base de datos: XXXX”

En la pantalla principal verá un botón con tres puntos en donde, si le hace clic, se mostrará un calendario para que elija la fecha necesaria para el proceso de facturas.

Si el sistema detecta que no eligió una fecha, al darle clic al botón de procesar, el sistema mostrará una alerta como la siguiente:

“Para procesar Facturas debe estar la fecha configurada”

Si entra al sistema para iniciar la operación de proceso sin haber cargado el directorio con los archivos CSV correspondientes, el sistema mostrará en pantalla un mensaje:

“No hay archivos para procesar”

Finalmente, si todo está correcto, al terminar de procesar, el sistema exhibirá los mensajes correspondientes, aparecerá la lista de archivos procesados y una leyenda que dice:

“Mensaje: los comprobantes se han registrado correctamente, Comprobante: FAC XXXXXX Estado: Ok”

Luego, en el sistema de TANGO GESTIÓN, en la ventana de Administración de comprobantes electrónicos, ya se podrán visualizar las facturas pendientes a obtener CAE.

Una vez que termine de procesar los archivos, es necesario moverlos a otro directorio para que no vuelvan a reprocesarse, ya que de ser así el sistema los detectará y emitirá los mensajes de validación.
